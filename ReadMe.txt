How to RUN: 
----------------------------------------------------------------------------------------------------------------
Go to Build Folder and double click Shadow Mapping.exe. Then Render->RunRender. 

Might take a few seconds depending on the maching for it to load the given input set of triangles and perform computations.

Creates an output.ppm image file.
